=== Meaning of export values ===

Value: Auto                                                                                                                                                                                                            
Meaning: The system automatically determines the export destination based on the entry's category/position. Rental-related items go to Hollgasse, tax-deductible items go to Arbeitnehmerveranlagung.                  
────────────────────────────────────────                                                                                                                                                                               
Value: Hollgasse                                                                                                                                                                                                       
Meaning: Forces export only to the Hollgasse rental property accounting Excel file (skips personal tax export).                                                                                                        
────────────────────────────────────────                                                                                                                                                                               
Value: Arbeitnehmerveranlagung                                                                                                                                                                                         
Meaning: Forces export only to the Austrian employee tax declaration Excel file (skips rental property export).                                                                                                        
────────────────────────────────────────                                                                                                                                                                               
Value: Both                                                                                                                                                                                                            
Meaning: Exports to both Hollgasse and Arbeitnehmerveranlagung files simultaneously. Used when a transaction is relevant for both rental accounting and personal taxes.                                                
────────────────────────────────────────                                                                                                                                                                               
Value: None                                                                                                                                                                                                            
Meaning: Entry is stored in the app but excluded from all Excel exports. For record-keeping without appearing in formal accounting/tax documents.
